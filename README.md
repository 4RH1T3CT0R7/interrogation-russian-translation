# Interrogation: You will be deceived — Русификатор

Полная русская локализация для игры **Interrogation: You will be deceived** (Critique Gaming / Mixtvision).

## Что переведено

- Все диалоги и допросы — 60 файлов локализации
- Главы кампании, эпизоды, интервью, катсцены
- Тесты персонажей — 30+ уникальных допросов
- Меню, титры, интерфейс
- Кириллические шрифты — 23 гарнитуры (включая системный шрифт)

## Установка

### Из релиза (рекомендуется)

1. Скачайте `InterrogationRussian-Setup.exe` из [Releases](../../releases/latest)
2. Запустите — установщик автоматически найдёт папку с игрой
3. Нажмите **Установить**
4. Запустите игру и выберите русский язык в настройках

### Структура после установки

```
Interrogation/
├── backup/                   # Резервные копии (создаются автоматически)
│   ├── game.arcd.original
│   ├── game.arci.original
│   └── game.dmanifest.original
├── game.arcd                 # ← пропатчен установщиком
├── game.arci                 # ← пропатчен установщиком
└── Interrogation.exe
```

## Удаление

**Через установщик:** запустите `InterrogationRussian-Setup.exe` и нажмите **Удалить** — оригинальные файлы восстановятся из бэкапа.

**Через Steam:** ПКМ на игре → Свойства → Локальные файлы → Проверить целостность файлов.

## Как это работает

Установщик патчит архив Defold Engine (`game.arcd` / `game.arci`) напрямую:
- Заменяет 60 записей локализации на русские `.ru.lua` файлы
- Заменяет 23 шрифта на версии с кириллицей
- Использует LZ4-сжатие, совместимое с движком
- Перед патчингом создаёт резервные копии оригинальных файлов

## Сборка из исходников

```bash
build.bat
```

Скрипт упакует данные в `data.zip` и соберёт установщик.
Результат: `InterrogationRussian-Setup.exe` в корне репозитория.

### Требования для сборки

- .NET SDK 9.0+
- PowerShell (для создания data.zip)
